# 🔥 Mad EZ Media — Fleet Financial Birdseye

Complete financial dashboard for Mad EZ Media's agent fleet operations.

## Pages

- **index.html** — Main dashboard with full fleet cost analysis, overdue invoices, desired stack breakdown
- **sonicbrand-cost-breakdown.html** — Detailed SonicBrand AI operational costs by service/API
- **ez360-cost-breakdown.html** — Detailed EZ Influencer 360 operational costs by service/API

## Data Sources

- Gmail cached inbox (May 11–19, 2026)
- ACMI Fleet (78 agents, 275 work items)
- Config files (50+ .env files across projects)
- Hermes CLI config

## Stack

- Pure HTML/CSS/JS — no build step, no dependencies
- Served via Vercel
- Dark theme, mobile responsive
